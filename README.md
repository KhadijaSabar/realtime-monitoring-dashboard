# Real-time Monitoring Dashboard

A real-time system monitoring dashboard that visualizes server metrics (CPU, RAM, Disk) with live updates and historical data analysis.

## Project Status

**Currently in development** - Expected completion: February 2025

## Overview

This project provides real-time monitoring capabilities for system resources with:
- Live metric updates via WebSocket
- Interactive charts and visualizations
- Historical data analysis
- Alert notifications for critical thresholds

## Tech Stack

### Frontend
- **React** - UI framework
- **Recharts** - Data visualization
- **Socket.io-client** - Real-time communication
- **Tailwind CSS** - Styling

### Backend
- **Node.js** + **Express** - REST API
- **Socket.io** - WebSocket server
- **PostgreSQL** - Metrics storage

### Data Collection
- **Python** - System metrics collector
- **psutil** - System monitoring library
